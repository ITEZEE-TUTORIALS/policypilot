# Alternate Topic: ManualMind

If you want a more consumer-facing version of the same RAG demo, use this framing:

**ManualMind: An Open-Book Support Assistant in Rust**

## Why this domain works
- It is easy for any audience to understand.
- Troubleshooting manuals are long and frustrating to search.
- The "look it up first" story is very easy to explain.

## Good example question
- "Why is my dishwasher showing error code E15?"

## Why it is still useful
- It shows the same RAG architecture as PolicyPilot.
- It is slightly more visual and product-like.
- It is a good fallback if the audience is not interested in business policy examples.
